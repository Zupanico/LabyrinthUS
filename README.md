# LabyrinthUS

Jeu d'horreur en C++ avec la manette Arduino

## Installation

```bash
git clone https://github.com/Zupanico/LabyrinthUS.git
```

## Usage

Exemples d'utilisation.

```bash
make
./labyrinthUS.exe
```
